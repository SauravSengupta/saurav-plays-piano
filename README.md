# Saurav Plays Piano 🎹

A minimalist ambient music portfolio built with **Astro** and **Tailwind CSS**. 
This site serves as a hub for my original piano compositions, specifically inspired by the Pacific Northwest.

## 🚀 Project Structure

- **Framework**: [Astro](https://astro.build)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Language**: TypeScript
- **Deployment**: Not sure yet, probably CloudFlare
